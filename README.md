# customer-behavior-analysis # Data Analytics Project

## Overview
This project demonstrates an end-to-end data analytics workflow using Python, SQL, Power BI, and Gamma.  
The project includes data loading, exploratory data analysis (EDA), data cleaning, SQL analysis, dashboard creation, reporting, and presentation development.

The goal of this project is to extract meaningful insights from raw data and present them in a clear and business-friendly format.

---

## Dataset
- Format: CSV / Excel / Database
- Size: 3900 Rows
- Description:
  The dataset contains information related to [sales/customers/finance/healthcare/etc.].

Example columns:

- Product_Name
- Sales
- Region
- Order_Date

## Project Steps

### 1. Data Loading
- Imported dataset using Python
- Loaded CSV/Excel files with Pandas
- Verified structure and data types

### 2. Exploratory Data Analysis (EDA)
- Checked dataset shape and summary statistics
- Identified missing values
- Analyzed trends and patterns
- Created visualizations and charts

### 3. Data Cleaning
- Removed duplicates
- Handled missing values
- Fixed inconsistent data
- Converted data types where necessary

### 4. SQL Analysis
- Imported cleaned data into PostgreSQL/MySQL/SQL Server
- Executed SQL queries for:
  - Aggregations
  - Filtering
  - Joins
  - Group By analysis
  - Business insights generation

### 5. Power BI Dashboard
- Built interactive dashboard
- Added KPIs and charts
- Created filters and slicers
- Visualized business insights effectively

### 6. Report Creation
- Summarized findings and insights
- Documented methodology and analysis
- Prepared final analytics report

### 7. Presentation Using Gamma
- Created professional presentation slides
- Highlighted project workflow and insights
- Presented dashboard screenshots and key outcomes
